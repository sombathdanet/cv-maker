# Quick Start Guide

## 🚀 Your Resume Builder is Ready!

The application is now running at: **http://localhost:5173/**

## What You Can Do

### 1. Fill Out Your Resume
- **Personal Information**: Add your name, title, contact details, and summary
- **Work Experience**: Click "+ Add Experience" to add jobs
- **Education**: Click "+ Add Education" to add schools
- **Skills**: Click "+ Add Skill" to add your skills

### 2. Live Preview
- Watch your resume update in real-time on the right side
- Professional, clean design similar to resume.io

### 3. Export to PDF
- Click the "📄 Export to PDF" button in the preview section
- Your resume will be downloaded as a PDF file

## Tips

- All fields marked with * are recommended to fill
- Use the "Currently working here" checkbox for your current job
- Choose skill levels: Beginner, Intermediate, Advanced, or Expert
- The preview shows exactly how your PDF will look

## Commands

```bash
# Start development server (already running)
npm run dev

# Build for production
npm run build

# Preview production build
npm run preview
```
